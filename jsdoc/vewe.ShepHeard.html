Class: ShepHeard

vewe. ShepHeard

new ShepHeard()

The ShepHeard is an object used for cross VEwe communication. All views in a view set have access to a shared ShepHeard and may subscribe, unsubscribe, and publish events to the ShepHeard in a fairly typical pub / sub fashion. In a sense, the ShepHeard shepherds the views.
